E-commerce Customer Purchase Behavior Analysis
📌 Project Overview

This project analyzes customer purchase behavior in an e-commerce dataset using Python. It focuses on identifying sales trends, top-selling products, and customer segmentation through data analysis and visualization.

🎯 Objectives
Analyze customer purchasing patterns
Identify top-selling products
Study monthly sales trends
Perform country-wise sales analysis
Segment customers based on spending
🛠️ Technologies Used
Python
NumPy
Pandas
Matplotlib
Google Colab
⚙️ Features
Data cleaning and preprocessing
Handling missing values
Feature engineering (Total Price, Date extraction)
GroupBy analysis
Sorting and filtering
Customer segmentation
Data visualization using graphs
📊 Output Visualizations
Top Selling Products (Bar Chart)
Monthly Sales Trend (Line Graph)
Country-wise Revenue (Bar Chart)
Customer Spending Distribution (Histogram)
📂 Dataset Information

The dataset used for this project contains transactional data of an online retail store, including:

Product details
Customer ID
Quantity and price
Invoice date
Country

⚠️ Note: The dataset is not included in this repository due to size and usage restrictions.

▶️ How to Run the Project
Open the project in Google Colab
Run the notebook
Upload your dataset file (CSV or Excel format) when prompted
The analysis and visualizations will be generated automatically
📈 Key Insights
A small number of products contribute to most sales
Sales vary across different months indicating seasonal trends
High-value customers generate a significant portion of revenue
Sales differ across countries
